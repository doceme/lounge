Lounge

Lounge

The Lounge module
The exports object of the lounge module is an instance of this class.
Most apps will only use this one instance. We copy all Couchbase Bucket methods and properties
so you can call them generically from this instance as well.

Connect to the database. You may define Models before connecting, but you should not create any actual document
instances before connecting to the database.

Example
lounge.connect({
  connectionString: 'couchbase://127.0.0.1',
  bucket: 'lounge_test'
});
